PRETZEL

Light-weight and portable linear structure playground

View on GitHub Download .zip Download .tar.gz

The foremost and primary goal of pretzel is to create a portable and minimalistic webGL visualization framework for 3D linear structures (i.e. biological chromosome). The library also provide a wide variety of utilities to interact with, examine, and simulate the structures in close details.

Features

High level abstraction
Simple and straightforward APIs
Simulation and data generator
